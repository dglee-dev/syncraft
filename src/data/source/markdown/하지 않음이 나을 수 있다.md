# 하지 않음이 나을 수 있다

'해야한다' 는 생각이 들어서 하는 일 말고,
'하고싶다' 는 생각이 드는 일을 차분히 고르고 생각해서 하려면

먼저 많은 방해들을 물리쳐야 한다.

더 혼자가 되어야 하고, 집중할 수 있어야 한다. 하나의 생각에 긴 시간동안 몰입할 수 있는 환경이 필요하다.
그렇게 큰 환경을 만든 후에야..내가 하고싶은 일에 대해 더 생각하고 고민하고 그것을 진행시켜 나갈 수 있음이다.

더 나답게..해나갈 수 있으려면, 시간이 필요하다. 공간이 필요하다. 이어지는 생각의 흐름이 필요하다.

그렇기 위해 사무실을 마련했다.
얼른 해야만 하는 일을 끝마치고..
내 일을 더 깊게 고민해 나가야겠다.

사람들보다 나와 내 생각, 내 감정에 더 귀 기울일 수 있어야겠다.

'~가 중요해! 지금 ~해야해!'

세상은 난리다. 챗GPT, 블록체인, ...

지금 안배우면 죽는 것처럼 난리들이다.
그게 필요하다는 생각이 '나에게 들면', 내가 적절하다고 생각하는 방식대로
그것들을 하나씩 내 삶에 적용시켜나가볼 수 있다.

그럴 때 적정 기술이 될 수 있다. 나에게 의미없는 기술은 살펴볼 가치가 없다.

그래서, 차라리 아무것도 하지 않는게 낫겠다는 생각이 든다.

그냥 먼저 좀 가만히 있어보자.
