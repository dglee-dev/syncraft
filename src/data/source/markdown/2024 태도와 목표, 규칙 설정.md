# 2024 태도와 목표, 규칙 설정

그간의 문제에 대해 먼저 생각해보자.

## 내 불행의 원천

생활을 단속하지 않은 것

지금은 가질 수 없는 가능성들에 마음을 빼앗겨 당장 -하기로 원하지만 그럴 수 없다는 것을 인정하지 못한 것

발을 들이지도 빼지도 않는 상태로 괴로워만 하는 것 - 마음가는 대로 결정해도 괜찮다.

## 규칙

- 생활을 단속할 것
  - 저녁에 일찍 자지 않으면 낮에 제대로 살아낼 수 없다
  - 낮에 제대로 살아내지 않으면 저녁에 제대로 잘 수 없다
  - 무엇 하나 제대로 하지 못한다. 불행해진다.
  - 관리되지 않는 시간에 신경쓸 것
  - 쾌락에는 우울이 따라온다
  - 운동하라

- 돈을 관리할 것
  - 한달에 100만원 저축할 것 (적금 활용)

- 여유로운 시간을 가질 것
  - ~를 해야 한다 는 생각에 자신을 몰아붙이지 말자
  - 더 중요한 것은 장기적으로 어긋나지 않고 나아가고 있는가 이다.
  - 꼭 지금 당장 하지 않아도 괜찮다. 정말로 괜찮다. 그냥 쉬어라. 분명히 나중에 할 수 있는 기회가 올 것이다.

- 장기적인 목표를 생각하고 현재에 집중할 것
  - 다음으로 네가 하고 싶은 일이 무엇인가
  - 그러려면 무엇을 해야하고 사람들에게 보여주어야 하나
  - 그렇다면 지금 당장 할 수 있는 작은 것을 완료해라

- 사람들과의 관계에 신경쓸 것
  - 있는 그대로 대하려 노력할 것
  - 신뢰를 형성할 것

- 죄책감 없이 몰입해라
  - 다른 일에 신경쓰지 않고 지금 하는 일에 완전히 몰입해도 괜찮다

- 죄책감 없이 일을 미뤄라
  - 하고싶지 않을 때에 그냥 하지 않기로 결정해버려라
  - 어영부영 미루지도 하지도 않기보다는 차라리 마음대로 결정해버려라.

- 시작할 시간을 정해라
  - 지금 관성적으로 하고 있는 것을 마무리하고 해야할 일을 시작할 시간을 정해라.
  - 5시부터 등산에 나선다!

## 무엇에 집중할 것인지

회사에서는 자생할 수 없고 상생한다.\
좋은 영향을 주고받으며 성장해야 한다.

나의 목표와 회사의 목표가 다르다는 것을 기억하자

회사에서의 나는 `믿음직한 막내 개발자` 역할을, - 공동의 목표 속 일원\
개인으로서는..`게임을 만드는 사람`으로.  - 작품 지상주의

그리고 지금은 둘 중 첫번째 역할이 더 중요하다.\
첫번째 역할을 잘 하지 못하면 두번째 역할의 기회 자체가 없다는 것을 기억해라

적어도 2024년에는 게임 만드는 사람으로서 본격적으로 시작할 생각은 하지 않는 것이 좋겠다. 이런 저런 게임들을 해보고 거기에서 뭐가 재밌었는지 기록하고 분석해보는 정도면 충분하다고 생각한다.

그리고 본업에 충실하면서 컴퓨터에 대한 이해를 키우고 수학을 공부하자.
