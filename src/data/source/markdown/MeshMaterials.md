# MeshMaterials

`MeshBasicMaterial`

=> This material is not affetct by the light.

`MeshStandardMaterial`

가장 기본적으로 사용할 수 있는 MeshMaterial 이다.
PBR (Physically based rendering)

`MeshDepthMaterial`

명암으로 깊이감을 확인할 수 있다.

`-Lambert-`

lambertian model을 사용한다.

`-Phong-`

blinn-phong model을 사용한다.
shiness, specular

`-Physical-`

Standard의 상위호환(?)
clearcoat: 거울과 같은 반사율 (transluent layer)
