# Vector intro

<iframe width="560" height="315" src="https://www.youtube.com/embed/br7tS1t2SFE?si=qJAtu4htpPGueYLd" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>

Vector has both magnitude(규모) and direction(방향).

예를 들어

- magnitude: 5mph (miles per hour)
- direction: east

5mph라는 속력(speed, scalar)과 east라는 방향이 만나 속도(velocity)라는 vector가 된다.
