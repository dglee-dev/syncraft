# 고정된 크기의 슬라이딩 윈도우

> Given an integer array `nums` and an integer `k`, find the sum of the subarray with the largest sum whose length is `k`.

subarray의 길이가 `k`로 제한되어 있다. 이전처럼 포인터를 활용하는 방법도 있지만, 이럴 때에는 k라는 크기를 이용해 합을 계산해나가며 sliding window 개념을 구현할 수도 있다.

```
function findLargestSumSubarray (nums, k) {
  // 0부터 k까지의 값들을 더해 가장 처음의 합을 계산한다
  let curr = 0;
  for (let i = 0; i < k; i++) {
    curr += nums[i];
  }

  // sliding window를 오른쪽으로 옮겨가며 가장 큰 값을 찾는다
  let ans = curr;
  for (let i = k; i < nums.length; i++) {
    curr += nums[i] - nums[i - k];
    ans = Math.max(ans, curr);
  }

  return ans;
}
```
윈도우의 크기가 동적이었던 투포인터는 포인터를 명시해야 하는 반면, 고정된 경우에는 윈도우의 크기(여기에서는 k)만을 가지고 계산을 이어나갈 수 있다. 코드에서 `i`와 `i-k` 를 이용해 왼쪽과 오른쪽의 값들을 빼고 더해나가며 합산이 유효한지 확인해나간다.
