# 과거회상 - 플룻

나는 어릴 때 플룻을 배웠다.

엄마는 왜 나에게 플룻을 가르쳤을까?
그건 꽤나 본인의 취향이 반영된 것이었을 것 같다. 우리 엄마는 클래식 음악을 들으며 집을 청소하고, 요리를 하고, 운전을 하는 것을 즐겨했고 교회에서도 클라리넷을 연주하는 사람이다.

지금 생각해보면 지극히도 본인 중심적이었구나 싶다. 나에게 '넌 뭐가 하고싶니' 라고 묻지 않았다는 것을, 어릴 때는 알지 못했다. 그저 엄마가 하라면 해야하는 줄 알았다.

초등학생 때 영민이라는 친구가 있었는데, 플룻을 배우러 백화점 문화회관에 가야해서 그 친구 생일날 가지 못했다.
다른 친구들이랑 시내에서 재미있게 놀았다는 이야기를 들었는데, 나는 그때까지 시내에서 친구들과 논다는게 뭔지 전혀 모르는 그야말로 집에만 있는 아이였다.

더 어린 시절에는 혼자나가 모르는 친구들과 딱지치기도 하면서 즐겁게? 낮가리지 않고 놀았던 기억이 있다.
그런데 언제부턴가 집에만 있는 아이가 되었고, 엄마가 시키는 것 아니면 그저 빈둥거리는 시간만 늘었던 것 같다.
방에 박혀서 눈칫밥을 먹으며 애니메이션을 보거나 게임을 하는게 나의 취미생활이었다.

과거를 떠올리면 항상 엄마나 아빠의 못난 모습들이 내 머릿속에 떠오른다. 왜 그렇게 해주지 않았을까, 왜 이렇게밖에 나를 조종하거나 방관하기만 했을까. 왜 내가 누구인지 궁금해하지 않았을까. 왜 내가 말하도록 허락하지 않았을까.

그들은 몰랐다.

그래서 어느순간부턴가, 부모의 정반대로 사는것이 정답이라 생각했던 것 같다.
그러나 그것조차 우스운것은 우리 아버지와 어머니는 꽤나 상극이다.

공통점을 찾자면 그들은 매우 자기중심적인 사람들이라는 것이다.

플룻으로 시작해 부모님을 욕하는 것으로 마무리되는 글이라..

나에게는 어떤 치유가 필요한 것 같다.
