# 구스노키 켄, 좋아하는 일과 감각

- 좋아하는 일을 해야 '감(감각, sense)' 을 쌓을 수 있다.
- 기술을 가진 사람은 많다. 나만의 가치는 감각에서 나온다.
- 감을 쌓기에는 작은 회사가 좋을 수 있다.
- 기술이 아닌 감각에는 '상대' 가 있다.
- 그렇기에 감각은 (상대에게) 결과로서 평가받는다.
- 감각을 갈고 닦는데에는 시간이 필요하다.
- 긴 시간동안 반복해서 무언가를 하려면, 그래서 감각을 쌓으려면, 그 일을 좋아하지 않고는 안된다.
