# 그냥 하지 마라

일단 하라는 말들을 한다.
고민없이, 일단 하고, 그것을 수정해 나가라는 말이다.

하지만 일단 하는 것이 좋은 경우가 있고, 그렇지 않은 경우가 있다. 2023년에는 대부분의 경우 아니다.

불모지에서 새싹을 심어 가꿔나가야하는 경우, 일단 하는 것이 유리하다. 아이폰이 처음 만들어졌을 때 일단 아무 앱이나 만들어보는 것은 매우 유리한 선택지였다. 지금 아무 앱이나 만들어서 시장에 내놓으면 아무도 사용하지 않는다.

세상은 점점 더 최적화의 길로 들어서고 있다. 이미 모든 것이 만들어져있다. 불모지를 개척하는 일은 더 이상 많지 않다.
최적화의 시대에 의미있는 일들은 크게 두가지이다.

1. 더 효율적이고 효과적으로 한다
2. 이미 있는 것들을 조합하여 새로운 가치를 만든다

두 가지 모두 '그냥 한다' 의 정신으로는 좋은 결과를 만들 수 없다.

어떻게 더 효과적으로 만들어낼지, 어떻게 새로운 가치를 만들어낼지 고민하고 실행해야한다.

그냥 마트를 차리면 안된다. 그냥 치킨집을 차리면 안된다. 그냥 해서 되던 시대는 한참 전에 지나갔다.
