# 그런 타입 몰라요! (TS 2307)

> Cannot find module '@icons/cross.png' or its corresponding type declarations.ts(2307)

TS2307 은 타입스크립트의 '그런 애 모른다' 는 하소연이다.\
..그런 애 모른다고 하니 친절히 알려주는 것이 우리의 할 일이다.

알려주는 방법은 [여기](https://www.typescriptlang.org/docs/handbook/modules.html)에 자세히 나와있는데, \
타입을 선언해주는 `d.ts` 파일을 작성해주면 된다. d는 declaration을 의미한다.

제목에서와 같이 png 파일을 인식하지 못하는 경우에는 아래와 같이 작성해주면 png 확장자를 인식하게 되어 내뿜던 에러메시지를 갈무리하는 모습을 확인할 수 있다.

* 타입스크립트가 인식하기 좋도록 `index.d.ts`를 프로젝트 루트경로에 위치시키는 것이 좋다. 그게 아니라면 tsconfig.json에 해당 파일의 위치를 명시해 주어야 한다.

```
// image.d.ts
declare module "*.png";
```
