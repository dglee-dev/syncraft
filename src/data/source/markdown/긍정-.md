# 긍정?

진실된 사람이 되자.  
건강한 사람이 되자.  
꾸준히 열심히 하는 사람이 되자.  
사람을 감동시키는 사람이 되자.  
어려움 속에서 작은 행운을 찾아내는 사람이 되자.  

자신을 속이지 않는 사람이 되자.  
멋져 보이려고 노력하지 않는 사람이 되자.  
인간다운 인간이 되자.  
어린 시절의 마음과 여유를 잃지 말자.  

세상에 영원한 내 것은 없다. 언젠가 나는 죽는다.
가지려고 애쓰지 말자.
