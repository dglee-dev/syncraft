# 내가 부자가 될 수 있는 방법

1. 돈을 더 번다
2. 돈을 아낀다
3. 모은 돈으로 돈을 더 벌 수 있는 방법을 마련한다

1.1 돈을 더 벌기 위해서는?

- 더 좋은 직장으로 이직한다.
- 또 다른 직업, 또는 사업을 가진다

1.1.1 더 좋은 직장으로 이직하기 위해서는?

- 지금 내가 하는 일들을 잘 정리해서 기록하자, 꾸준히

1.1.2 어떤 다른 직업, 또는 사업을 가질 수 있을까?

- **게임, 여행, 패션, 음악을 올리는 유튜브를 하자**
- 너무 특별하지 않은 편한 영상들을 만들어서 올려보자
