# 대순형과 사운드 비주얼라이저 앱(?) 아이디어

- 우리가 잘하는 것이 무엇인가?
- 음악적인 완성도는 낮겠지만
- 이 음악을 새로운 방식(나)으로 예쁘게(형) 전달할 수 있다
  - 새로운 방식?
    - App (=앨범)
    - 인터랙티브 비주얼
    - 수민의 그것
  - 예쁘게
    - 예쁘게!

- 사운드를 고를 수 있다
  - 사운드를 조절할 수 있다
  - 스플라이스에서 사운드샘플 구매
- 각각의 사운드는 이미지로 표현된다
  - 사운드에는 태그들이 달려있을 것 (스플라이스 등)
  - 해당 태그를 기반으로 정해둔 비주얼이 표현된다
- 사운드를 쌓아나가는 것이 이미지의 레이어로 쌓여나가는 것으로 표현된다
- 작곡과 시각적 이미지를 동시에 얻으며, 어떤 도움을 주고받으며 결과물이 만들어질 수 있다

- 이 2차 창작물은 오롯이 본인의 것이기 때문에!
- 자유롭게 이 안에서 공유될 수 있고 심지어 판매될 수 있다
- 각각의 저작권이 NFT로 관리된다!
