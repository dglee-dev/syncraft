# 무엇이 나를 움직이는가

나를 움직이게 만드는 것들은 무엇인가?

새로운 개념. 새로운 분야. 처음 시작하는 일. 작은 것을 내 손으로 직접 만드는 일. 남들이 잘 하려고 하지 않는 일. 보지 못한 것들 보려고 하는 것..온통 마이너 힙스터 아싸 총집합이구먼.

나는 어린 시절 나름의 리더쉽도 있었고 인기도 있었던 기억이 있다. 근데도 항상 혼자만의 즐거움은 따로 있었던 것 같고. 집에서 듣는 음악, 만화책, 게임들.

혼자만의 즐거움들은 나를 채워준다. 그 에너지를 토대로 사람들에게 예상치 못한 즐거움을 선사하는 것을 즐긴다. 누군가 시키는대로 움직이기보다는 창의적인 방식에 대해 평등하게 소통하여 좋은 결과물을 내놓는 일에 관한 것인 편이 좋다.
