# 수학 책들

내 언젠가는 이것들을 모두 읽을테다.

첫번째 책 Discrete Mathematics with Application 은 [이 글: Logic and Multiply-Quantified Statements in JavaScript](https://adueck.github.io/blog/logic-and-multiply-quantified-statements-in-javascript/)에서 알게되었다. Adam Dueck이라는 엔지니어가 쓴 글인데, quantified statements라는 수학적 개념을 어떻게 자바스크립트에서 발견하고 사용할 수 있는지에 대한 글이다.
