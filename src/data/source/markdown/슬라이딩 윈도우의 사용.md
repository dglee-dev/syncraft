# 슬라이딩 윈도우의 사용

> You are given a binary string `s`. \
You may choose up to one `"0"` and flip it to a `"1"`. \
What is the length of the longest substring achievable that contains only `"1"`? \
\
For example, given `s = "1101100111"`, the answer is 5. If you perform the flip at index 2, the string becomes <ins>11111</ins>00111.

1이 가장 길게 이어지는 substring을 찾는 것이 이 문제의 핵심이다. 그런데 '1이 가장 길게 이어지는' 을 다른말로 하면, '최대 1개의 0을 포함하는' 으로도 바꿀 수 있다. 우리는 1개의 0을 1로 바꿀 수 있도록 명령받았기 때문.

> 최대 1개의 0을 포함하는 가장 긴 substring을 찾아라

어디에서 많이 본 문제같아지지 않았나?

기존에 '1이 가장 길게 이어지는 - 찾아라' 라는 표현에서는 직관적으로 O(n^2)의 브루트포스 알고리즘을 떠올리게 되는 반면, '최대 1개의 0을 포함하는' 이라는 형식으로 표현하게 되었을 때에는 해당 조건을 만족하는 subarray를 찾아야겠다, 즉 투포인터를 이용할 수 있겠다는 생각이 들게 된다.

```js
function findLength (str) {
  let left = 0;
  let numberOfZero = 0;
  let ans = 0;

  for (let right = 0; right < str.length; right++) {
    if (str[right] === "0") numberOfZero++;
    
    while (numberOfZero > 1) {
      if (str[left] === "0") numberOfZero--;
      left++;
    }

    const subArrayLength = right - left + 1;
    ans = Math.max(ans, subArrayLength);
  }
}
```
코드에서는 가장 긴 길이를 반환하고 있지만 left, right로 잘라낸 subarray(substring)을 반환할 수도 있을 것이다.
