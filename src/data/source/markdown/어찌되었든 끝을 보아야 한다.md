# 어찌되었든 끝을 보아야 한다

지금 나는 제대로 마무리짓지 않은 일이 마음에 남아있어 불쾌하다. 그래서 생각해보았는데, 이 불쾌함과 걱정들을 없애려면 다음과 같은 원칙이 필요하다. 

> 한번 시작한 일은 어찌되었든 끝까지 한다.

끝이 무엇인가를 설정하는 일도 중요해질 것이다. 무쪼록 무언가를 만드려는 생각으로 키보드든 붓이든 펜이든 잡았다면, 그것을 어찌되었든 `끝까지` 한다. 얼마의 시간이 걸리든 끝까지 하는 것이 중요하다. 그러므로 빠르게 마칠 수 있다면 빠르게 걱정을 덜어낼 수 있을 것이다만 그것이 마음대로 되지는 않는다.

내가 잘 못하는 것, 끝까지 하는 것. 어찌되었든 끝은 봐야한다. 어찌되었든, 끝을 봐야한다. 그것이 좋은 끝이 아닐지라도, 그것이 좋을지 아닌지는 끝을 보아야 알 수 있는 것이기 때문이다.

끝을 보자.
