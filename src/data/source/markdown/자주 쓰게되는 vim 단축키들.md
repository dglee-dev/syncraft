# 자주 쓰게되는 vim 단축키들

### 네비게이션

- 파일 끝으로 커서 옮기기: `shift + g`
- 문장 끝으로 커서 옮기기: `shift + $`
- 문장 처음으로 커서 옮기기: `shift + 0`

- 단어 단위로 앞으로 이동: `w`
- 단어 단위로 뒤로 이동: `b`

### 입력/삭제

- INSERT 모드에서 단어 하나 지우기: `control + w`
- INSERT 모드에서 라인 지우기: `control + u`
- 되돌리기: `u`
- 되돌리기 취소: `control + r `
