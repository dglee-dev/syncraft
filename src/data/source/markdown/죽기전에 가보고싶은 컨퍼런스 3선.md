# 죽기전에 가보고싶은 컨퍼런스 3선

## 2. Ars Electronica Festival

오스트리아 린츠에 위치한 문화과학재단 아르스 일렉트로니카의 페스티벌이다.
'다학제적 뉴 미디어 아티스트를 발굴 및 지원하는 것이 목적' 인 기관의 페스티벌인 만큼, 뉴 미디어를 통해 다양한 주제들을 다양한 방식으로 표현하는 작품들이 주를 이룬다. 2024년에는 9월 3-4일에 개최.

## 3. TGS (Tokyo Game Show)

도쿄 게임 쇼. 게임은 일본게임이다. 가지각색의 게임들을 한 자리에서 만나볼 수 있을 것.  
다만 여름(보통 9월에 개최)에 일본이라..사람도 많을텐데 괜찮을지 모르겠다.

2024년에는 9월 28-29일 일반 공개. 비즈니스는 26-27일.
