# 혼자 일하고 싶다면 무엇을 준비해야 하는가?

내 최고의 목표는..

「혼자 일하는 사람이 되는 것」

혼자 일하려면?

회사가 가진 인프라들을 가져야한다.

첫 번째로.. 「고객」
두 번째로.. 「동료」
마지막으로.. 「사무실」

세 가지 중 가장 중요한 것은 고객이 아닐까.

내가 지금 가질 수 있는 고객은 누굴까?
누군가에게 무언가 가치있는 것을

「지속적으로」

제공할 수 있다면..
그가 나의 고객일 것.

작은 크기의 고객을 유치한다.
작은 크기의 고객을 심도깊게 유지한다.

그리고, 「팬」 으로 만든다.

그렇게 팬 베이스가 생기면, 직장을 그만두고 규모를 늘린다.
