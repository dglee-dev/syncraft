# 회사 생각들

그가 방어적인 이유

- 인정받지 못한다는 마음?
  - 먼저 그가 한 일들을 인정하자


우리가 재미없는 이유

- "해야한다" 는 눈치로 일하기 때문
- 주도적으로 일하려면, 무엇이 어떻게 바뀌어야 할까.
- 내가 한 일이 어떤 의미가 있는지 충분히 이야기한다
  - 조금 여유롭게 하자
  - 여유로운 시간에 할 수 있는 일들
  - 차 마시기
  - 내 일에 대해 충분히 이야기하기
    - 하지 않아도 될법한 이야기를 하기
    - 어떤 디테일들을 신경썼는지에 대해 이야기하기



더 넓은 범위에서..

- "잘 해내고 있다" 라는 것을 시각적으로 보여준다
- 증명한다
- 여기에서 내가 할 수 있는 일들이 많겠군! 싶다.
- 전체 회의때 더 자세하게 보여주고, 이야기해주자.
